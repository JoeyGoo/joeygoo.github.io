Empty reply from server

当hexo deploy时实际调用的是git指令去push到github(或其他地址上)，今天执行hexo deploy时出现了标题中的错误,如下

fatal: unable to access ‘https://github.com/JoeyGoo/joeygoo.github.io.git‘: Empty reply from server;

后来查明原因，发现是github上HTTPS通道被墙的原因，访问失败，改成SSH后就可以了，点击查看如何配置SSH通道
